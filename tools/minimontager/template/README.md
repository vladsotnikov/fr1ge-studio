# Монтажер 1.0 — AI Video Assembly Kit

Автоматична збірка відео з AI-кліпів + озвучки. Без Premiere, без After Effects.

## Що це робить?

Бере твої відеокліпи + аудіо-озвучку → автоматично нарізає кліпи під тривалість кожної фрази → склеює → накладає звук → видає готове відео.

## Швидкий старт

### 1. Встанови залежності
```bash
brew install ffmpeg
```
Потрібен Python 3 (вже є на Mac).

### 2. Підготуй файли
```
template/
├── input/
│   ├── voiceover.mp3          ← твоя озвучка (ElevenLabs, записана, будь-яка)
│   └── clips/
│       ├── 001-video.mp4      ← кліп для сцени 1
│       ├── 002-video.mp4      ← кліп для сцени 2
│       └── ...                ← і так далі
├── project.json               ← конфіг (опис сцен)
└── scripts/
    └── montage.py             ← скрипт монтажу
```

### 3. Заповни project.json
```json
{
  "audio_file": "input/voiceover.mp3",
  "output_file": "output/final_video.mp4",
  "resolution": "1280x720",
  "clips_dir": "input/clips",
  "scenes": [
    {
      "id": 1,
      "clip": "001-video.mp4",
      "text": "Текст озвучки для першої сцени"
    },
    {
      "id": 2,
      "clip": "002-video.mp4",
      "text": "Текст озвучки для другої сцени"
    }
  ]
}
```

**Головне правило:** поле `text` — це ТОЧНИЙ текст, який диктор говорить під час цієї сцени. Скрипт рахує слова і розподіляє час пропорційно.

### 4. Запусти
```bash
python3 scripts/montage.py project.json
```

### 5. Забери результат
```
output/final_video.mp4
```

## Як працює монтаж?

1. Визначає тривалість аудіо (ffprobe)
2. Рахує слова в кожній сцені
3. Розподіляє час пропорційно: більше слів = довший кліп
4. Кліп коротший за потрібне? Лупиться (повторюється)
5. Кліп довший? Обрізається
6. Все склеюється в один відеоряд (FFmpeg concat)
7. Накладається озвучка
8. Готово

## Поради

- **Роздільна здатність:** `1280x720` (горизонтальне) або `1080x1920` (вертикальне/Shorts)
- **Кліпи:** будь-який формат який читає FFmpeg (mp4, mov, webm)
- **Озвучка:** mp3, wav, m4a — що завгодно
- **Мова:** працює з будь-якою мовою (рахує слова, не розпізнає мову)
- **Кількість сцен:** без обмежень, тестовано на 177

## Формат project.json

| Поле | Обов'язкове | Опис |
|------|:-----------:|------|
| `audio_file` | так | Шлях до озвучки |
| `output_file` | ні | Шлях для результату (default: `output/final_video.mp4`) |
| `resolution` | ні | Роздільна здатність (default: `1280x720`) |
| `clips_dir` | ні | Папка з кліпами (default: `input/clips`) |
| `scenes` | так | Масив сцен |
| `scenes[].id` | так | Номер сцени |
| `scenes[].clip` | так | Ім'я файлу кліпа |
| `scenes[].text` | так | Текст озвучки для цієї сцени |
