# Voice Stock Studio

Web-додаток для:
- завантаження озвучки;
- автоматичної транскрипції;
- підбору релевантних відео/зображень по контексту;
- монтажу фінального ролика (MP4) під озвучку;
- роботи в двох режимах: `Stock (Pexels)` або `Local files`.

## Швидкий старт

1. Встановити залежності:
   ```bash
   npm install
   ```
2. Запустити сервер:
   ```bash
   HOST=127.0.0.1 PORT=3000 npm start
   ```
3. Відкрити:
   ```
   http://127.0.0.1:3000
   ```

## Як робити монтаж

1. Завантажити аудіо.
2. Натиснути `Розпізнати озвучку`.
3. Натиснути `Підібрати медіа`.
4. Натиснути `Змонтувати відео`.
5. Після рендера з'явиться кнопка завантаження готового `mp4`.

## Ключі API

- Для транскрипції потрібен `OpenAI API key`.
- Для stock-режиму потрібен `Pexels API key`.

## Вимоги

- Node.js 18+.
- `ffmpeg` має бути встановлений у системі та доступний у PATH.
