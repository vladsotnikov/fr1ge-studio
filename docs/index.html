🍎 Встановлення на Mac

  1. Скачай відповідний DMG зверху (Apple Silicon або Intel — залежить від моделі).
    Як дізнатись який у мене Mac?

    Apple Menu → About This Mac. Якщо у "Chip" написано M1/M2/M3/M4 — бери Apple Silicon. Якщо Intel — бери Intel-версію.

  2. Подвійний клік на скачаний файл — відкриється вікно з аппкою.
  3. Перетягни іконку 🟣 Fr1Ge STUDIO у папку Applications.
  4. Йди в Applications → правий клік на Fr1Ge STUDIO → Open.
    ⚠️ macOS пише "невідомий розробник" — це нормально

    Аппка не підписана Apple-сертифікатом ($99/рік не платимо для приватного кола). Тому при першому запуску треба:

    • Правий клік (або Ctrl+клік) на іконці аппки → Open
    • Натиснути Open у попередженні системи

    Тільки перший раз. Далі можна запускати звичайним подвійним кліком.

  5. З'явиться splash з полем для ключа — введи свій ліцензійний ключ. Якщо нема ключа — попроси у адміністратора.
  6. ✅ Все. Аппка готова до роботи.

🪟 Встановлення на Windows

  1. Скачай Windows-інсталятор зверху.
  2. Подвійний клік на скачаний `.exe`.
  3. З'явиться попередження SmartScreen "Windows protected your PC".
    ⚠️ Це нормально — як обійти
    • Натисни "More info"
    • З'явиться кнопка "Run anyway" — тисни її

    Тільки перший раз.

  4. Інсталятор покаже стандартний wizard — натисни "Install".
  5. Після встановлення з'явиться ярлик Fr1Ge STUDIO на робочому столі.
  6. Запусти аппку → введи ліцензійний ключ у splash → готово.

🧠 BLIP (локальний AI-аналіз)

BLIP потрібен тільки якщо ти використовуєш "Файли з компʼютера" як джерело медіа і хочеш щоб AI підбирав найкращий кадр під кожен сегмент озвучки.

  1. У головному вікні Fr1Ge STUDIO зайди на вкладку Автопілот
  2. Обери джерело "Файли з компʼютера"
  3. Зверху побачиш картку "🧠 Локальний AI-аналіз (BLIP)"
  4. Натисни "Завантажити BLIP (~1.5GB)"
  5. Прогрес-бар покаже стани: Python → PyTorch → BLIP моделі. Це займе 5-15 хв залежно від інтернету.
  6. Готово — BLIP працює офлайн назавжди (не потребує перезавантаження після оновлення аппки).

Без BLIP працює: транскрипція, стоки, рендер монтажу, нарізка YouTube. BLIP просто додає AI-підбір кадрів у локальному режимі.

❓ FAQ

Чому в мене питає ліцензійний ключ?

Аппка призначена для обмеженого кола користувачів. Адмін видає ключі індивідуально. Ключ зберігається на 7 днів — після цього перевірка повторюється.

Скільки коштує?

Аппка безкоштовна. Витрати — тільки OpenAI API (на твій ключ ~$0.05-0.20 на ролик) та опціональні Pexels/Pixabay ключі (безкоштовні tier-и).

Чи зберігаються мої файли десь у хмарі?

Ні. Все рендериться на твоєму компʼютері. Аудіо/відео не залишають твій Mac/PC. На сервер йдуть тільки запити до OpenAI/Pexels (з твоїми API-ключами).

Що робити при помилці рендеру?

Аппка має resume-from-cache: якщо рендер упав на кліпі 47/60 — наступний запуск з тими самими інпутами продовжить з того місця. Просто перезапусти монтаж.
